# Configuração do Banco de Dados
1. Instale o MySQL ou PostgreSQL em seu ambiente local.
2. Crie um banco de dados chamado `meu_banco`.
3. No arquivo `.env`, insira suas credenciais:
   DB_HOST=localhost
   DB_USER=seu_usuario
   DB_PASSWORD=sua_senha
   DB_NAME=meu_banco
4. Execute o projeto!
